# Senseforce
Frontend Test Automation Assignment

## Installation | Node Js

```
brew install node
```

or

Download from official site https://nodejs.org/en/

## Installation | Testcafe

```
npm install
```

## Usage

```
npm run test             # run all tests in chrome headless mode
npm run test-chrome-gui  # run all tests in chrome using GUI
```
